# 오늘의 포켓몬은 뭘까요 🔴

포켓몬 이름을 입력해서 도감을 채우는 퀴즈 게임.

- 전 세계 포켓몬 **1025종** (메가진화 · 리전폼 · 거다이맥스 포함)
- 이름을 맞히면 해당 포켓몬의 모든 폼이 한꺼번에 열림
- 세대별 · 타입별 필터, 실루엣 모드, 스트라이프 릴 지원
- 진행 기록은 브라우저(localStorage)에 자동 저장

## 실행

`index.html` 파일 하나로 완전히 독립 동작. 모든 이미지(1,498장)가 파일 안에 포함되어 있어 인터넷 연결 없이도 실행된다.

## GitHub Pages로 배포

1. 이 폴더 내용을 저장소에 업로드
2. 저장소 **Settings → Pages → Branch: main / (root)** 선택 후 저장
3. 잠시 후 `https://<계정명>.github.io/<저장소명>/` 에서 플레이
