# 오늘의 포켓몬은 뭘까요 🔴

포켓몬 이름을 입력해서 도감을 채우는 퀴즈 게임.

- 전 세계 포켓몬 **1025종** (메가진화 · 리전폼 · 거다이맥스 포함 1250칸)
- 이름을 맞히면 해당 포켓몬의 모든 폼이 한꺼번에 열림
- 세대별 · 타입별 · **도감별(18종)** 필터, 실루엣 모드, 모습 자동 전환(한 포켓몬의 여러 폼 자동 순환) 지원
- 진행 기록은 브라우저(localStorage)에 자동 저장

## 실행

`index.html` 파일 하나로 완전히 독립 동작. 모든 이미지(1,498장)가 파일 안에 포함되어 있어 인터넷 연결 없이도 실행된다.

## GitHub Pages로 배포

1. 이 폴더 내용을 저장소에 업로드
2. 저장소 **Settings → Pages → Branch: main / (root)** 선택 후 저장
3. 잠시 후 `https://<계정명>.github.io/<저장소명>/` 에서 플레이

### 핵심 데이터 구조 (단일 `<script>` 내)

- `DATA` — 전체 엔트리 배열. `[이름, 국가도감번호(0-index), 타입1, 타입2, 박스번호, 이미지ID배열, ...]`
  - `e[1]` = 국가도감번호-1, `e[4]` = 박스(세대 구간) 인덱스
- `BOXES` — 박스 정의: 0~5=1~6세대, 6/7=메가XY/ORAS, 8=7세대, 9=세대불명, 10=8세대SwSh, 11=거다이맥스, 12=8세대LA, 13/14=9세대(본편/DLC), 15/16=메가LZA
- `mb(b)` — 박스 병합(14→13, 16→15), `DISP` — 박스 표시 순서
- `DEXDEF` — 도감 버튼 정의 `[이름, 필터함수]` (배열 20개, 메뉴 표시는 18개). 인덱스: 0전국 1관동 2성도 3호연 4신오 5하나(번호 0000부터) 6칼로스(통합·3섹션, 7·8은 구 코스트/마운틴으로 메뉴 숨김+dexIdx 리맵) 9알로라 10가라르 11갑옷섬 12왕관설원 13히스이 14팔데아 15북신 16블루베리 17미르 18이차원 19메가진화(전용 모드, MEGA_DEXNO 번호·엔트리 단위 카운트)
- `*_NATDEX` — 각 지역도감의 공식 종 목록(국가도감번호 1-index, 도감 등재 순서). 출처: 성도=HGSS, 호연=RSE, 신오=Pt(210), 하나=B2W2(301), 칼로스=XY 3분할, 가라르/갑옷섬/왕관설원/히스이=Bulbapedia, 알로라/팔데아/북신/블루베리/미르/이차원=PokeAPI
- `DEXORDER` — `{DEXDEF인덱스: NATDEX.map(n=>n-1)}`. 이 목록이 있으면 도감 구성·순서·번호 모두 이걸 따름. 없는 도감(메가진화)만 DEXDEF 필터 사용
- `VARIDX` — 리전폼/메가/거다이맥스 변형 엔트리의 표시 토글(cfg.regional/mega/gmax) 매핑
